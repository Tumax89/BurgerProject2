React-ийн онцгой философи
Реактийн онцгой архитетур, философи
1. UI Library- React нь фрэймворк биш харин UI Library
Том системийг хэрхэн үр ашигтай хийхийг мэднэ.
Реакт нь UI хэсгийг хурдтай, үр ашигтай байлгана
Дэлгэцийн мэдээлэлийг хэрхэн хадгалах
дэлгэцийг хэзээ UPDATE хийхийг
Дэлгэцийн хаана нь UPDATE хийхийг  React хариуцдаг.
Дэлгэцийг бүтээх кодыг JSX ашиглан маш хялбараар бичих боломжийг React өгдөг.

2. Single source of Truth буюу үнэний нэг эх үүсвэр
Өгөгдөл нь доошоо дамждаг, Event-үүд нь дээшээ дамждаг.

3. Өгөгдөлийн нэг чигт урсгал - Uni direction data flow

Ерөнхий санаа: Ямар ч өгөгдөлийг дамжуулах цор ганц зам байна.

React-ийн санаа:
Ямар нэг төлөвийг зөвхөн нэг л компонент дотор л хадгална.
Компонентийн төлөвийн өөрчлөлт зөвхөн доод буюу доторх компонентууд руу л дамжигдана.
Компонентүүд нь ҮЙЛДЭЛ/ACTION хийнэ
Үйлдэл нь төлөвийг өөрчилнө.
Төлөвийн өөрчлөлтийг компонент болон түүний доторх компонентууд руу дамжуулна.

4. Declarative, REACTIVE, virtual DOM
 Declarative - зарлан програмчлах зарчим
 REACTIVE - хариу автомат риакцийн зарчим
